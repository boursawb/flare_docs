Layout PrintWhen expression

All layout stamps can be configured with a PrintWhen expression to conditionally display the stamp based on a set of defined criteria. This is accomplished by adding a small snippet of JavaScript to the stamp in question. In the example shown below, a text stamp has been configured with a PrintWhen expression that will ensure the stamp is only applied if the signature contains a slate. This is accomplished by inspecting the file name of the signature and if it matches the criteria defined, the stamp is applied. 

<printWhen>
	<script>
		<![CDATA[
		
		// Lookup the stamp data in the map
		var fileName = data.fileName;
		
		// If null, do not treat as a slate
		if (fileName == null) {
		return false;
		}
		// Otherwise, if filename has slate in it, treat it like a slate
		else {
		return fileName.contains('slate-front');
		}
		]]>
	</script>
</printWhen>

All code from the opening <printWhen> tag to the closing </printWhen> would typically be placed prior to the first stamp attribute or following the last stamp attribute for ease of readability.

Defining a PrintWhen expression

In order to configure a stamp to use a PrintWhen expression, the following tasks must be completed.

  • Identify the stamp requiring the conditional logic
  • define the logic and Identify the attributes that will be used to generate the conditional statements
  • Add the javascript to the stamp
