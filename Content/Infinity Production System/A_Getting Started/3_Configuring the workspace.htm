Configuring the Workspace

provides several options for configuring the workspace to meet your needs. Depending on your role within the organization, the workspace can be setup to show information that is most applicable to your job duties.

Adjusting Columns

One of the main areas of customization within IPS is determining which attributes (columns) will display on a given tab. From any tab, you can add, remove, re-size and reposition columns as needed.

Moving columns
  1. Locate the column to be moved
  2. Click and hold while dragging the column to the new position
  3. Let up on the mouse button to complete the action
 Add new columns

There are two methods for adding new columns to the UI. Columns can either be added one at a time or multiple columns can be added by using the customize columns feature.

Add single column

  1. Change to the tab where the new column should be added
  2. Right-click on any column
  3. Locate the column from the list and select it

The list of available attributes will extend far beyond the size of the screen. There is a small arrow at the bottom of the list that when hovered over, will scroll the list down.

Add multiple columns

  1. Change to the tab where the new column should be added
  2. Right-click on any column
  3. Select the Customize columns... in the top section of the menu
  4. In the selection dialogue, check the box next to each column that should be included in the view
  5. Click OK to save the changes and close the dialogue
 Remove columns
  1. Change to the tab where the new column should be added
  2. Right-click on the column to be removed
  3. Select the Hide This Column option at the top of the menu

Multiple columns can be removed un-checking the box next to the items which should be removed from view.

Sorting columns

The column data in the user interface can either be unsorted (how they arrived in the system), sorted ascending or sorted descending. How the information sorts depends on the data type of the information the column contains. For example, if a column contains a 7 digit number but the data type for that column is of type string, when sorted, the column will sort alphabetically, not numerically. Additionally, IPS allows you to perform a multi-column sort by clicking each column in the desired order while holding the CTRL key down. The order of the grouped columns will be indicated by a small number on the column header.

Sorting columns
  1. Click the column header to initially sort the column ascending
  2. If descending order is preferred, click the column header once more
  3. To return the column to an unsorted order, click the column header a third time
  4. To sort multiple columns together, while holding down the CTRL key, click column A, then column B followed by column C, etc.

A triangle in the corner of the column header indicates the sort currently applied to the column. A triangle pointing up indicates ascending sort order, and a downward pointing triangle indicates descending. When multiple columns have been CTRL+clicked, and superscript number will show on the column to indicate its ranking.

Example - Sorting multiple columns

Assume you have the columns Created, Received, and Updated in your UI. If you wanted to sort all of the data first by Created, then by Updated, and finally by Received, you would hodl the CTRL key down and then click each of the column headers in the order previously mentioned. This will result in a superscript 1 on the Created column, a 2 on the Updated column, and finally a 3 on the Received column.

Resizing columns

Any column in the UI can be re-sized to ensure all data is visible. Columns can either be re-sized manually or automatically.

Manual resizing
  1. Hover the mouse over the vertical line on the right hand side of the column to be re-sized
  2. Wait for the horizontal double-arrows to display and then hold down the mouse button and drag to the right
 Automatic resizing
  1. Right-click on any column header
  2. Select either the Re-size this column or Re-size all columns option from the top of the menu

Filtering column data

IPS provides the ability to filter the UI to display only data that you are interested in. This can be useful when only needing to see a very specific set of information. For example, it may be desirable to display only orders for a given customer, only specific products, or simply display orders that have encountered an error and need immediate attention. Filtering a column in the UI allows you to display only rows with a given value.

Filtering a column
  1. Hover the mouse over the right side of the column to display the filter icon
  2. Click the filter icon and select the specific item from the drop-down list to filter by
 Creating a custom filter
  1. Hover the mouse over the right side of the column to display the filter icon
  2. Click the filter icon and select Custom... from the list
  3. Select the condition to be included in the filter
  4. Select the value (if applicable) for the condition
  5. Click OK to apply the filter to the column
