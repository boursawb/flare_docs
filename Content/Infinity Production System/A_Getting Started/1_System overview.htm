IPS core components

The Infinity Production System (IPS) is built on the following core components.

Distributed Coordinator

The Distributed Coordinator will run on one server within the production group and typically resides on the Master Core Host. This service ensures that all components defined to be in a cell are the only ones running in that cell. Additionally, the coordinator will ensure that two workers never take on the same work at the same time. For example, it is likely that a site will have the need to run both a stage and production environment. Each environment will include a distributed coordinator, one or more Distributed Core (DCore) servers and an order injector server. It is the job of the distributed coordinator to keep these environments independent and running in their own space.

Distributed Core Processor (DCore)

Each server will run as a Distributed Core Processor, with the exception of the database server. Among several other activities, the DCore is responsible for moving orders through the various phases of th eIPS workflow. The main workflow steps that each order will go through include inject, analyze, prebatch, sort, and batch.

Order injector (OI)

The order injector is the entry point for orders into the system. Standard order injection is handled by monitoring a folder where XML files (orders) can be picked up and processed. Custom plugins can be built through Infinity Professional Services that allow IPS to seamlessly integrate with third party / customer order management systems.

User Interface (UI)

The user interface is the customer's view into the production workflow. From the UI, you can view order information currently in the system, identify what has been released to press and not, and view completed batches and press jobs.

Integration

offers several points of integration into the Infinity workflow. There are two primary points of integration. The first, briefly mentioned above is the order injector. also integrates with several industry digital front end devices (DFEs). The current DFE integrations standard in IPS include the following.

  • HP Indigo
  • Kodak Nexpress
  • Xeikon
  • Xante
  • Netgate
  • Network hotfolder

Jobs can also be submitted to standard network printers using the Infinity Digital OS Print Service solution.

Infinity software developers kit (SDK)

The Infinity software developer's kit provides detailed information on building an appropriately structured XML file that can be submitted to a a folder for order submission. The SDK provides a standard approach for building orders in XML format that can be generated by several industry standard tools and order management systems.
