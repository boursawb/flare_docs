Thumbnail stamps

There are two forms of thumbnails that can be generated when building product templates. The first method involves using the imagestamp attribute of a layout to extract an image from a specific page of the order content PDF. The second method involves parsing a URL that extracts an image from a defined page of the PDF order content. The second option applies to thumbnails stamped on related content and additional spreads that have been created with Jaspersoft Studio.

Stamping thumbnail images using layouts

Generating thumbnails from the layout can be done by using a specific syntax for the imagepath attribute of an image stamp. To specify the page of the PDF content to use as the thumbnail you will use the syntax shown below (taken from a layout image stamp).

imagepath="%thumb_10080_pg2%"

The three attributes shown above between the percents signs include thumb to designate that we are generating a thumb stamp, 10080 which denotes the product we are interested in, and pg2, which specifies which page of the PDF the system will pull the image from. When a product has more than one component, the layout being used will determine which component the thumbnail will be generated for.

Example - Multi-component product

Assume we are working with a book product that consists of a book and cover component. Each component defined within prebatch will have its own associated layout file. If you add the imagestamp code in the layout associated with the book, the thumbnail generated will be for the book component. If the stamp information is placed in the layout associated with the cover, the thumbnail generated will be from the cover's content.

It is important to note that only a select number of pages are available for thumbnail stamping. In most situations, you will be able to stamp pages 1-5 and n-3 of the order content.

Stamping thumbnail images via URL

When building related content with Jaspersoft studio, thumbnails are generated by telling the system to pull an image from a specified location using a URL. the URL is comprised of the order id, line id, component id, and page number. To add a thumbnail to a report in Jaspersoft Studio, follow the steps shown below.

  1. Drag an image container onto the pallet where the thumbnail will be displayed
  2. Set the value of the image expression to the use the desired image
