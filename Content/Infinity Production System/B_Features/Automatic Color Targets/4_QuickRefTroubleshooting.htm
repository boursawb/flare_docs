Troubleshooting

Some common issue encountered when working with the color quality feature in include the following.

  • The device name listed in the press numbers section of the barcode settings does not match the actual device name specified in the DFE configuration file
  • Any of the related XML configuration files contain a parsing error
  • Incorrect project or location code specified on the Site Information tab of the barcode settings
  • All intended products have not been included in the product bin. This will result in the number of spreads counted being less than expected
  • The incorrect component was specified in the product bin for a given batch definition
  • The target description file was not created or does not contain the required contents
  • The file name of the color target (TIF file) and the color target definition file name do not match
  • A .template.xml layout file matching the name of the color target has not been created
  • The color target file name is incorrect as specified in the target ID field within the product bin
