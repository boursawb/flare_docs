Color target submission

Color targets can be submitted manually from the UI or can be sent automatically when a defined threshold has been exceeded. For detailed information on target thresholds, refer to "Understanding target thresholds" on page 1.

Manual target submit
  1. Select the Color quality tab within the UI
  2. Click the generate color target button
  3. Select the desired product bin
  4. Select the desired DFE / Device to submit the target to
  5. Click the Generate button
  6. Click the Close button
 Automatic target submission
  1. Verify the product bin contains all batch components to be tracked
  2. Verify the target threshold has been set to the desired value
  3. If using light volume threshold, set the desired amount of time and ensure Indicate Start of Shift has been clicked
  4. Save the product bin

For each batch component contained within the product bin that results in a job showing printed within , the counter for the product bin / press combination will be incremented by the number of spreads contained within the job. Once the total number of spreads for this counter has been exceeded, a color target will be submitted to press.
