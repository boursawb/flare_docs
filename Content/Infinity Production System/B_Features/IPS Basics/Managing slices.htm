Managing slices

To manage existing order slices, administrators can use the IPS Production Admin web interface. To access the order slice page, browse to the url shown below.

http://<yourServer>:8384/ips_production_admin#/orderslice

In the above url, <yourServer> must be the host name of a server running the distributed coordinator (DCore) process. If you have questions about which server that is, contact your system administrator or Infinity Support.

From the IPS Production Admin, the following order slice actions can be performed.

  • View the current slices
  • Filter the list by a specific key word
  • Create a new slice (must use an existing matcher, cannot build one from this screen)
  • Export the current list of slices to a .csv formatted file
  • Edit an existing slice

To edit an existing slice, follow the steps shown below

  1. Click the EDIT button to the right of the slice
  2. Make any necessary changes
  3. Click the Save button
