Quick Statistics

Quick stats provide a way for displaying key pieces of data that are meaningful to your organization. How and what information you choose to display is completely up to you. Quick stats can be found in the window located at the bottom of the UI.

Before creating your first quick stat, let's take a look at the types of data that can be displayed and the options you have available to you when building them. Some examples of common quick stats may include...

  • Total number of orders
  • Total number of orders for customer ABC
  • Orders not batched
  • Total errors in sort
  • Number of orders meeting a defined threshold (turns red)

Quick Statistics Interface

The quick stats interface consists of four tabs which provide various configuration options that can be used during the creation process. The following section discusses each of these tabs.

Filter

Select a column allows you to select a column that will be used to filter out only the data you are interested in displaying.

Condition specifies the condition that will be applied to the filter. The items that display in the condition drop-down will change depending on the type of data that is selected in the Select a column drop-down. Once a condition has been defined, you must add it to the filter list. You can add as many conditions to the filter list as needed, but it is important to note that all of the conditions must be met for the data to display (AND operand).

 Summary

Summary Column In the summary column drop-down, you will select the data item that you wish to display in the quick stat.

Summary Type allows you to select how this data should be displayed. The values available in this drop-down will may vary depending on the value selected in the summary column drop-down.

 Title

Title Specify the title to be used for the quick stat. Title is a required field and must be filled in prior to saving the quick stat.

Font Style Allows you to apply basic font styles of bold and italic

Font Enables the ability to select a specific font to display for the quick stat title

Background Specifies the background color to be used for the quick stat

Foreground Specifies the font color to be used for the quick stat

Horizontal Alignment Specifies the horizontal positioning of the text within the quick stat

 Conditional Styling

Preview Displays the area at the top of the conditional styling tab that displays what the quick stat is going to look like

Condition Allows you to apply a condition that must be met for the particular style to be applied

Style for the condition Sets the font, font style, background, foreground and horizontal alignment that will be applied to the quick stat data value when the condition has been met

Make sure to add the new conditional style to the List of conditional styling box by clicking the Add to List button. If your conditional style does not display in the List of conditional styling box at the bottom of the screen, your style settings will not be saved.
