Batching from sort groups

For most production environments, the majority of the time spent in the UI will be on the sort groups tab. The sort groups tab provides the main interface for releasing work to press, provides statistical information for work not yet released, allows you to change the active sort profile, and much more. There are two methods for releasing work, manual device selection and automatic device selection. When manual device selection is used, you will choose the press to submit the batch to by selecting the desired destination from the batch dialogue. When automatic device selection is used, the job will be batched based on the current device group configuration.

Releasing Batches

From the sort groups tab, there are multiple options for submitting work to press.

Release Many Batches

With release many, you can release all batches, a specific number of batches, or release all batches with a priority date older than the selected value. The release many option also allows you to choose manual or automatic device selection during the process.

  1. Highlight the sort group(s) to perform the action on
  2. Right click on the highlighted row / selection
  3. Select the option to Release Many Batches
  4. Choose the desired option and click Next
  5. Select the option for manual device selection or automatic device selection and click Next. If manual device selection was chosen, select the destination and click Finish

Manual Release Next Batch

This option presents the batch dialogue where you will select the destination device for each component being submitted to press.

  1. Highlight the sort group(s) to perform the action on
  2. Right-click on the highlighted row / selection
  3. Select the option to Manually Release Next Batch
  4. Click Yes on the confirmation dialogue to release the batch from the sort group
  5. Select the desired destination on the batch dialogue and click Ok

Release Next Batch

Releasing the next batch from the sort group will result in the top-most batch in the group being released to press as defined by the device group configuration. No option to manually choose the destination will be displayed.

  1. Highlight the sort group(s) to perform the action on
  2. Right-click on the highlighted row / selection
  3. Select the option to Release Next Batch

The batch will be submitted to the appropriate press as defined by the device group configuration.

Manual Release All

Manually releasing all results in all batches contained with the highlighted sort group(s) to be released at once and allows you to select the destination device the batches should be sent to.

  1. Highlight the sort group(s) to perform the action on
  2. Right-click on the highlighted row / selection
  3. Select the option to Manually Release All batches
  4. Click Yes on the dialogue to confirm the batch release
  5. Select the destination device for all components and click Ok

Release All

This option releases all batches from the sort group(s) to the destination device associated with the configured device group.

  1. Highlight the sort group(s) to perform the action on
  2. Right-click on the highlighted row / selection
  3. Select the option to Release All batches
  4. Click Yes on the dialogue to confirm the batch release

The batches will be submitted to the appropriate press as defined by the device group configuration.
