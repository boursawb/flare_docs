Batching from products

Creating batches can be accomplished from the products tab, the sort groups tab, and the batches and press jobs tabs by reprinting existing batches. As most work will typically be released from the sort groups tab, there will be times when creating one or more batches from the products tab will be required. A typical scenario that may require a batch to be created from the products tab might be for an order that needs expediting and cannot wait in line behind other orders in the batch. When creating batches from the products tab, there are two options, Create Batch and Create Single Product Batch.

Create Batch
  1. Highlight one or more orders on the products tab
  2. Right-click on the highlighted order / selection
  3. Selection the option to Create Batch
  4. Select the destinations for the batch component(s) on the batch dialogue
  5. Click Ok
 Create Single Product Batch
  1. Highlight one or more orders on the products tab
  2. Right-click on the highlighted order / selection
  3. Selection the option to Create Single Batch
  4. Select the destinations for the batch component(s) on the batch dialogue
  5. Click Ok
  6. When creating single product batches, will create a unique batch for each item selected on the products tab, basically resulting in a one-to-one mapping between orders batched and batches created.
