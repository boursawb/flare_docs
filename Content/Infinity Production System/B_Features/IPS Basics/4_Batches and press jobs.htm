Batches and press jobs

For each successful batch submitted, one or more press jobs will be generated. For batches that include more than one component, multiple press jobs will be created. A batch reprint can be performed from either the batches or press jobs tab. One additional option to Reprint batch as single order per batch exists on the batches tab and can be used to create a one-to-many output from the original batch. Both tabs contain a column displaying status lights to indicate which step of the given work flow the batch or press job has reached. The status lights can be used to diagnosing why a batch has failed to complete or job has not been successful in making it to press. It is important to understand that the press job status lights display the status of down-stream devices. IPS will display job status information only if an integration exists for the given device.

Reprinting batches

The following batch reprint options can be performed from the batches tab within

Manual Reprint Batch

This option allows you to select the destination device for each component to be reprinted. In the case of a batch containing multiple components, you can remove the check from any component which should be excluded.

  1. Highlight the batch to be reprinted
  2. Right-click on the batch
  3. Select the option to Manually Reprint Batch
  4. Click Yes on the confirmation dialogue to reprint the batch
  5. Select the desired destination on the batch dialogue and click Ok

Reprint Batch

Reprinting the batch will resubmit the job to press using the device group configuration to determine the appropriate destination for the job.

  1. Highlight the batch to perform the action on
  2. Right-click on the highlighted batch
  3. Select the option to Reprint Batch

Reprint batch single order per batch

This option will result in n number of new batches being created, 1 for each order within the original batch.

  1. Highlight the batch to perform the action on
  2. Right-click on the highlighted batch
  3. Select the option to Reprint single order per batch

This option for batching jobs to press is less common than the others but can be invaluable in determining which order in a batch has resulted in a failure. When a single order fails to submit, it will result in the entire batch failing. This option can be used to create individual batches for each order allowing the majority of the work to get to press and at the same time, highlight the order which resulted in the initial failure.

Manual reprint press jobs (Press jobs tab only)

The Manual Reprint Press Jobs feature streamlines the process for reprinting one or more press jobs associated with an exception batch. To reprint specific jobs from an existing exception batch, follow the steps shown below.

  1. Change to the Press Jobs tab
  2. [CTRL + click] each press job that should be included in the reprint (each will be highlighted)
  3. Right-click on any of the highlighted jobs and select the Manual Reprint Press Jobs item from the menu
  4. Click Yes on the confirmation dialog to perform the reprint
  5. Confirm the batch dialog has all expected boxes checked based on your selection
  6. Click OK to perform the reprint

The press jobs selected must be part of the same batch. Selecting jobs from two different batches may produce unexpected results or a failure during batch processing.
