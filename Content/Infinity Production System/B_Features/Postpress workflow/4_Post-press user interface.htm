Post-press user interface

When an item has been submitted to the post-press workflow, a job will display in the UI on the post-press tab. The post-press job is created immediately after the press job is generated.

Suggested post-press UI columns

  • Workflow state
  • Finishing device
  • Finishing device input hotfolder
  • Product Type
  • Batch Component
  • Post press workflow created start

Post-press jobs do not get cleaned up automatically. It is the responsibility of the finishing operator to manually delete jobs from the post-press tab once they are no longer needed.
