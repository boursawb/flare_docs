Auto-Release

Auto-release lets you configure to submit batches of work with minimal user intervention. In most situations, operators will decide which batches are released, to what device, and when. At times, there may be a desire to use the auto-release feature to all IPS to make these decisions automatically, as work enters and processes through the system. Auto-release relies on rules that you configure to determine when a batch should be released from the system, and uses device groups to determine which device the batch should be released to. All of this can be accomplished with no interaction from the operator, once configured. In order to use auto-release, the following items must be completed..

  • Any sort group that will use auto-release must have a release criteria defined
  • Every release criteria has a valid matcher assigned to it
  • Auto batch is enabled in the selected release criteria
  • Auto Release is enabled on the sort groups tab
  • An appropriate release filter has been defined (if implemented)
