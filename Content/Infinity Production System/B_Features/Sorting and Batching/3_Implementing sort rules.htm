Implementing sort rules

Once all rules have been created, any additional maintenance or modifications needed can be performed from the Sort Rule Configuration interface. There are 3 steps which must be performed following the creation of sort rules.

  1. Add the rules to the appropriate sort profile
  2. Specify the order of rules within the sort profile
  3. Set the active sort profile

Adding rules to a sort profile

Only rules that exist in the active sort profile will be used during the order sorting process. When adding rules to a sort profile you will add all rules currently defined in the system and remove those that are unwanted.

  1. Click the All Sort Profiles link in the Configuration Objects window
  2. Click the active sort profile in the main search window so it is highlighted, right-click and select Modify Sort Profile

  1. Click the Add all rules to this profile link
  2. Once all rules have been added, click the minus button to remove any rules that should not exist in the profile

Ordering rules within a profile

Given that an order can only have a single rule applied, it is critical that the order of rules within the sort profile is very deliberate. In general, you will order the rules from most strict to least strict, with a final catch-all rule at the bottom. Setting the active sort profile

The active sort profile determines which rules will be in play. At any given time, there can only be one active sort profile. You can configure as many sort profiles as needed, each with their own rule definition and you can switch between sort profiles to meet business requirements. The following steps illustrate how to change the active sort profile.

  1. Change to the sort groups tab
  2. Click the button to Change Sort Profile
  3. Select the desired profile
  4. Click Ok
