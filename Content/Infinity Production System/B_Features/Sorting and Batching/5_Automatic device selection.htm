Automatic device selection

Automatic device selection occurs when any of the normal release options have been selected by an operator. A normal release can be performed from the sort groups or batches tab and includes any of the following actions.

  • Release many batches / automatically choose device (sort groups)
  • Release next batch (sort groups)
  • Release all (sort groups)
  • Reprint batch (batches tab)

Upon performing one of these actions, will submit the batch to the configured device group. For additional information related to configuring device groups, refer to Device groups.
