Press Rooms

Press rooms provide a way to logically organize remaining work in the system by specifying a device group on desired batch components and then associating those device groups to a press room. Once configured, you can view the total sheet and spread counts not yet batched for one or more press rooms. Press room statistics can be viewed from the Sort Groups tab by clicking the Press Room Stats button. A typical Press Room configuration might include a press room for Books, one for Large Format, and a third for Envelopes. All products that contain the desired batch components for books would be configured to use a device group that would be added to the Book press room. All sheet and spread count data, broken down by press room, could then be viewed from the sort groups tab in .

The above illustration shows 3 press rooms and their associated sheet and spread counts. Checking the box next to a press room will add the sheet and spread counts to the sort group summary information displayed to the right of Press Room Stats button. Enabling or disabling a press room has no impact on the rows displayed on the sort groups tab. Press rooms simply determine which products will be included in the overall sheet and spread count.

Key terminology

Familiarity with the following terms is recommended when working with press rooms.

  • Estimated Spreads: The number of spreads in the largest batch component (press job) across all product batches contained within a sort group. For example, if the sort group contains a 2 component batch (book block and cover) and the largest component is the book block.
  • Estimated Spreads Total Batch Spreads: The sum of all spreads within all product batches contained in a sort group. Essentially, the total number of spreads contained in all batches within the sort group
  • Batch Component: A batch component is contained within a product batch and equates to a press job
  • Spread: A single side of a press sheet (for simplex products) and each side of the press sheet (for duplex products)
  • Sheet: The entire sheet (both simplex and duplex products)

Press room overview

The following image shows three press rooms and the device groups associated with each. Given the imposition used by each batch component, calculates the total number of sheets and spreads required to release the work to press.

As orders complete the sorting process, batch components are inspected to determine if there is an associated device group. then adds the total number of spreads within that batch component to the appropriate press room. For example, if a batch component has been configured to use the device group called CAN24x36, the number of spreads generated by the batch component will be added to the Large Format press room. In the event a batch component has not been configured to use a specific device group, it will resolve to the default device group for the batch component's destination type.

Creating a press room

To create a new press room, follow the steps shown below.

  1. Navigate to the \\<masterCoreHost>\idf\sharedconfig\dfe\pressroom
  2. Create a new <name>.pressroom.xml file (i.e. Envelope.pressroom.xml - example shown below)

<pressRoom systemReservedId="">
	<deviceGroups>
		<deviceGroup>5.75 x 4.375 - ENVL</deviceGroup>
		<deviceGroup>6 x 6 - ENVL</deviceGroup>
		<deviceGroup>7.25 x 5.25 - ENVL</deviceGroup>
	</deviceGroups>
</pressRoom>

  1. Add the device groups that will be associated with this press room
  2. Save and close the file

IMPORTANT - A device group can only exist in one press room. In the event a device group is added to multiple press rooms, only the first press room the device group is found in will be counted. It is also important to note that device group names defined in press rooms are case sensative and must match the case of the associated device group.

Working with press rooms

To display all press rooms, click the Press Rooms Stats button at the top of the sort groups as shown below.

The above illustration shows 3 defined press rooms. Books and LargeFormat both show accumulated sheet and spread counts as determined by the associated device groups. The summary information to the right of press rooms accounts for the sheet and spreads across all enabled press rooms. Disabling (removing the check mark) a press room will reduce the total spread and sheet count shown in the summary information by the amount for that press room.

NOTE: Specific press room selections can be saved to a UI perspective for future use.

Enabling press room columns

Press rooms can be displayed as a sort group column by performing the steps shown below..

  1. Right-click on one of the columns
  2. Choose Customize columns from the menu
  3. Check the box next to each press room that should be displayed as a column on the press jobs tab

Only press rooms which contain spreads greater than 0 will display as available columns to add.

Device group to press room map

Clicking the Press Room Stats button will display a link below the last press room called Device group to Press room map. The press room map displays a list of all device groups defined in the system and shows which press room each device group is associated with. A blank press room column indicates that the device group has not been added to any press room.
