Creating sort rules

The Sort Rule Configuration Tool can be used to create and manage rules defined in the system. The left window pane allows you to filter the view to only the items you wish to see and the right window pane provides the ability to create each of the different order sort components. When working with sort rules, keep the following in mind.

  • Orders can only have one rule assigned to them
  • Rules are applied in order, as defined by the active sort profile
  • Only one sort profile can be active at any given time
  • Sort rules group orders together
  • Split by lists divide sort groups following a sort rule match

Sort rule configuration wizard

Rules can be created manually, or by using the Create New Sort Rule wizard. The following steps will guide you through the process of creating a new sort rule using the wizard.

Creating a new rule

  1. Launch the Sort Rule Configuration management console
  2. In the Actions pane, click the link to Create New Sort Rule

  1. Click Next on the initial overview screen
  2. Specify a unique and meaningful rule name and click Next

  1. Choose the appropriate sort group from the drop-down and click Next. If no sort groups currently exist or a new group is required, click the link to create a new sort group.
  1. Specify the name of the new sort group and click Ok.

  2. Select the appropriate matchers and click Next. If no matcher exist, or a new matcher is required, click the link to create a new matcher

  1. Provide a unique and descriptive name for the new matcher
  2. Select either Order or Product from the drop-down list based on the types of attributes required. If you are unsure which level to select, you use the view order attributes web page on the products tab for one of the orders and determine which level should be used
  3. Select the attribute from the drop-down list to match on
  4. Select the desired operator
  5. Specify the desired value
  6. After being returned to the wizard, click the + to add the newly created matcher to the rule
  7. Check the box next to the desired matcher(s) and click ok
  8. Click Next to continue to the Split by list screen
  9. Select the appropriate split by list from the drop-down menu and click Next. To create a new split by list, click the Create a new split on attribute list link

  1. If a new split by list will be created, follow the steps below. If you are using an existing split by list, continue from step 21.
  2. Provide a unique and descriptive split by list name
  3. Add as many split by list attributes as desired and click Ok
  4. Select the split by list created in the previous step and click Next
  5. Click Finish

The purpose of sort groups is to provide a level of organization and grouping for the sort rules created. Additionally, sort groups specify the release criteria that will be used when auto-release has been enabled.
