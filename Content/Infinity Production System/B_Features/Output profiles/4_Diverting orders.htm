Diverting orders

Once all configurations have been made and the alternate profile products built, you are ready to divert orders from the products and sort groups tabs as needed.

Diverting from the products tab
  1. Locate and select the order(s) on the Products tab to be diverted
  2. Right-click anywhere on the selection and click the Divert to alternate output profile menu option
  3. Select the desired output profile from the drop-down list
  4. Click the Ok button to divert the orders

 Diverting from the sort groups tab
  1. Locate the sort group to be diverted
  2. Right-click on the top-level of the sort group and click the Divert to alternate output profile menu option
  3. Select the desired output profile from the drop-down list
  4. Click the Ok button to divert the orders

  5. When diverting a large sort group, it will take some time for all orders to process completely before they will be ready to batch to the intended output profile.

Order history

By viewing order history, you can see if an order was diverted and the output profile that was selected..

The divert action is written to order history when the user clicks on the divert to alternate output profile menu item, not following the divert process. This means that you will see the divert entry in the order history even if you simply select the menu option and then click the cancel button. For this reason, it is important to look for the reprocess actions following the divert entry to determine if the order was successfully diverted.
