Creating alternate profiles

Once you have built the output profile definition, you are ready to identify products that will use the new output profile. A product can have as many output profiles defined as needed. A product will be defined as an alternate output profile if its prebatch and batch definitions contain the outputprofileid attribute. In order to successfully divert a product to one of its alternate output profiles, the following four conditions must be met.

  • The batch and prebatch definitions for the product requiring the alternate profile must be copied to new files with names different from the originals.
  • The batch and prebatch definitions for the alternate profile must include the outputprofileid attribute
  • The value in the outputprofileid attribute must match exactly between the batch and prebatch definitions
  • An output profile definition must be created with its name matching the value of the output profile ID specified in the batch and prebatch definitions

Creating a new output profile

Creating an alternate output profile for a specific product involves the following actions.

Duplicate the product

  1. Open the batch definition for the identified product in an XML editor such as Notepad++
  2. Add the attribute outputprofileid="yourProfile" to the IDFBatch section of the batch definition
  3. Perform a Save As on the file and append _yourProfile to the end of the file name prior to the .XML extension
  4. Add the attribute outputprofileid="yourProfile" to the IDFPrebatch section of the prebatch definition
  5. Perform a Save As on the file and append _yourProfile to the end of the file name prior to the .XML extension

Create the output profile definition

  1. Browse to the sharedconfig\outputprofile\outputprofiledef folder on the master core host (database server)
  2. Create a new definition with the name yourProfile.outputprofiledef.xml
  3. Open the file and put the <outputprofile></outputprofile> tags in it
  4. Close and save the file

Viewing alternate output profiles in the configuration browser

Once the new output profile has been created for the product, it is a good idea to view the new product in the configuration browser. From here, you will be able to see the default profile for the product and also validate that the new profile was created successfully.

View in Config Browser

  1. View product in configuration browser
  2. Locate the product on the Products tab of the UI
  3. Right-click on the product and select Product Configuration Browser

Alternate output profile products will show up in the configuration browser with their output profile ID in parentheses to the right of the product name.
