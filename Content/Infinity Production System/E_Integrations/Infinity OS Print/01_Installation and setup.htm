Installation and setup

The Infinity Digital OS Print service can be installed by running the current release of the self-extracting installer. Once the installer completes, you will be able to access the OS Print web interface by clicking the launcher icon from the Windows start menu, or opening a web browser and pointing it to http://yourHost:9300, where yourHost is the name of the server where the OS Print Service was installed. Each feature of OS Print requires that you have a valid license installed before that feature can be used. The features included in the current release of OS Print are shown below.

  • OS Local Print Device
  • Scodix Foil Device
  • JPG Hotfolder Device

Installation

  1. Obtain the latest release of the OS Print Services installed from Infinity Digital
  2. Copy the installer to the server that will run the OS Print service
  3. Double-click on the installer package
  4. Accept all defaults
  5. Complete the installation
  6. Open the Windows services applet and stop Infinity - OS Print Service
  7. Double-click the service to open the properties
  8. Click the Log on tab
  9. Select This Account and specify an account that the OS Print Service will use. Make sure this account has sufficient privileges to write to the osprint staging location, device staging locations, and final output locations as discussed in the configuration section.

Installing dependencies

The OS Print service provides two methods for rendering content submitted to one of the previously mentioned devices.

Ghost Script

Ghostscript is an interpreter for PostScript™ and Portable Document Format (PDF) files. The print quality of the rendered output is sufficient for related content and batch time additional spreads but may not be suitable for customer content if submitted to a Windows print device.

Download and installation

  1. Open a web browser to http://ghostscript.com/download/gsdnld.html and click the link to download the Windows 64 bit edition
  2. Save the file to the Windows desktop
  3. Double-click the file to launch the installation wizard
  4. Accept all defaults during the installation

Adobe Rip

This option converts PDF files to a format suitable for sending to Windows print devices. If content sent to a Windows printer requires a high resolution output, it may be necessary to use this option instead of Ghostscript for rendering the output.

Download and installation

  1. After acquiring a license, contact Infinity Digital Support to have PDF2IMG installed in your environment.

The use of PDF2IMG will incur additional licensing fees.

OS Print system configuration

  1. Log in to the machine where the OS Print service was installed
  2. Identify a drive with sufficient space to hold the OS Print staging folder
  3. Create a new folder on the drive identified in the previous step called osprint
  4. Right-click on the folder, click Sharing, click Advanced Sharing, check the box to Share this folder and leave the default share name of osprint
  5. Click the Permissions button and add the account the OS Print service was configured to use with Change permissions (minimum), preferably Full Control
  6. Create a new folder within the osprint folder previously created called staging
  7. Launch a web browser and point it to localhost:9300
  8. In the left navigation menu, click Settings
  9. Click System within the Settings list
  10. For job storage local path, enter the full path to the staging folder created above (i.e. \\<yourHost>\osprint\staging)
  11. For job storage UNC path, enter the same UNC path as shown in the previous step (\\<yourHost>\osprint\staging)
  12. Set the value for Delete jobs older than to 1h (This is how long you would like completed jobs to remain in the OS Print web interface)
  13. Leave the default (2) for Rip pool worker size
  14. Click the Save button to apply the changes

Device licensing

Before you can create or use any OS Print devices, a valid license must be loaded via the web interface. If you have purchased a license for OS Print but have not yet received it, contact Infinity Digital Support for further assistance. If you have received your license, it can be loaded by following the steps shown below.

  1. Click About in the left navigation menu of the OS Print web interface
  2. Click on License at the top of the screen
  3. Click the Load License link
  4. Paste in the license key you received via email
  5. Click the Apply button in the lower right corner of the browser

If successful, you will be presented with a page that shows your company information, along with the features provided by your license.
