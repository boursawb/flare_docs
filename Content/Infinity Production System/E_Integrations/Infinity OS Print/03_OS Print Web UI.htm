OS Print web UI

Management and administration of OS Print is done through a web user interface. From here you can configure OS Print, create and configure devices, update license information, as well as view and manage jobs currently in the system. The web UI can be accessed browsing to http://<yourHost>:9300 where <yourHost> is the name of the server that the OS Print Service was installed on.

The recommended / supported browser for accessing the Infinity OS Print application is Google Chrome.

When opening the OS Print web UI, you will see a screen similar to the following.

From the Dashboard, you can perform the following actions.

  • Create Job
  • Devices
  • Jobs
  • Settings
  • About

Create Job

The create job screen allows you to manually submit a job via the OS Print web interface. To manually submit a job, follow the steps shown below.

  1. Select a device from the Devices drop-down list
  2. Click on the hotspot and browse to the file(s) to upload
  3. Select the Settings and Advanced tabs and make any required adjustments (Not all devices will have settings to change)
  4. Click the Save button

Devices

From the Devices page you can view, edit, create, and delete OS Print devices. Any existing devices will be displayed as soon as the Devices page loads. To create a new device, simply click the Create button in the upper right corner of the page. To edit a device, click on the device from the Device List. To delete an existing device, click the device to edit it, then click the trash can icon located in the upper right corner of the page. Fro detailed information on how to configure a new device, see Creating a new device

Jobs

The jobs page displays all jobs currently in the system. From here you can see when the job was received, how long it took to rip, and the current status. Clicking on a job will take you to the job details which displays additional information about the job that can be used for detailed troubleshooting and analysis. Once a job has completed processing, it will be automatically removed from the jobs list following the configured retention time. In most environments, it is recommended to set the Delete jobs older than to 4h (4 hours).

Settings

All OS Print system level configurations will be made within the Settings menu. From here you will configure the following system parameters.

  • Job storage local path
  • Job storage UNC path
  • Delete jobs older than
  • Rip pool worker size

For detailed information regarding the configuration of OS Print, see Installation and setup

About

Here you will find a brief description of OS Print, along with the version number for the instance of OS Print that is currently installed. This is also where you will go to view or update the license used by OS Print.
