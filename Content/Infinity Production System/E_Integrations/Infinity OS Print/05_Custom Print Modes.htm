Custom Print Modes

From the OS Print web interface, you can create custom print modes for submitting jobs to a Scodix device. A custom print mode consists of the following attributes.

  • Print Mode Name The identifier for the print mode used in the submit job dialog
  • TIFF X/Y Resolution The values for the X and Y resolution of the TIFF
  • SJD Print Mode Id Override An override for the print mode id that is placed into the SCODIX job definition format file (SJD). In this example, FOIL53ALIAS would show up in the SJD file instead of FOIL53. If no value is provided the Print Mode Name is used instead.

Adjusting built-in print modes

In situations where the values for a built-in print mode must be changed, a new print mode will need to be created that uses an alias that matches the desired print mode name, with adjustments being made in the newly created print mode. For example, if the required print mode is Foil 50 but the vertical resolution required is 1270 instead of 635, a new print mode would need to be created with the desired vertical resolution and an alias name matching the required print mode name. To create a custom version of a built-in print mode, follow the steps shown below.

  1. Identify the print mode to be used (i.e. Foil 50)
  2. Click Scodix Print Modes under Settings within the OS Print web UI
  3. Click the Create button in the upper right corner
  4. Enter a new name for the print mode (i.e. Foil 50V2
  5. Enter the required values for horizontal and vertical resolution
  6. Enter the desired print mode name for the alias (i.e. Foil 50)
  7. Save the new print mode

Creating a custom print mode

To create a Scodix custom print mode, follow the steps shown below.

  1. Open Chrome and browse to the host running the OS Print service - http://<yourHost>:9300
  2. Click Scodix Print Modes under Settings
  3. Click the Create button in the upper right corner
  4. Enter the value for Print Mode Name
  5. Enter the value for TIFF X resolution
  6. Enter the value for TIFF Y resolution
  7. OPTIONAL - Enter the value for the SJD Print Mode ID Override
  8. Save the new print mode

Edit / Clone an existing print mode

To create a new print mode from an existing, or modify one or more values of a custom print mode, follow the steps shown below.

  1. Click either the EDIT or CLONE buttons to the right of the print mode
  2. If editing an existing print mode, make the required changes and click save
  3. If cloning a print mode, provide a new print mode name, make any necessary changes, and click save

Delete a custom print mode

To delete an existing print mode, follow the steps shown below.

  1. Click Scodix print modes under the Settings menu
  2. Check the box for the print mode(s) to be deleted
  3. Click the DELETE button in the upper right corner
