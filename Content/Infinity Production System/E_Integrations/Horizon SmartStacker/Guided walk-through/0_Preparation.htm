Preparation

A key step in this process will be to collect several pieces of data that will be used to complete the setup. To simplify this process, much of the data can be pulled directly from configuration files you have likely already built.

From the batch imposition

The following data points can be obtained from the batch imposition.

  • Sheet width The value specified for spreadwidth
  • Sheet height The value specified for spreadheight
  • Vertical gutter The value specified for vgutter
  • Horizontal gutter The value specified for hgutter
  • Rows The number of rows defined as specified in the signatures section of the imposition
  • Columns The number of columns defined as specified in the signatures section of the imposition

Determining gutter requirements

If the total space between the trim marks of adjacent pages is not greater than or equal to 4mm (11.339 points), an amount will need to be specified for the appropriate gutter to make up the difference. For example, if the imposition we are working with has a value of 0 for both horizontal and vertical gutters, and the trimout required is 4.5 points on all sides, then the total distance between our signatures after the trim occurs would be 9 points. This is 2.339 points below the minimum required so we would add the appropriate values to our vertical and horizontal gutters to make up the difference

From the customer

The trimout required will be determined by you, or your customer. This is the amount of space that will be removed from each side of the product during the finishing process. Four attributes are provided for controlling the amount of trim on each side, but in most situations, the same value will be used for all four to ensure the product is trimmed equally on all sides.

  • Trim left The amount in points that will be removed from the left side of each signature
  • Trim bottom The amount in points that will be removed from the bottom of each signature
  • Trim right The amount in points that will be removed from the right side of each signature
  • Trim top The amount in points that will be removed from the top of each signature

From prebatch output

When building the laycrimp layout file we will need to know the full size of the product that will be fed into the imposition. We can get the product width and height by generating prebatch for an order and then inspecting the page size of the PDF that is generated.

  • Product width The width of the product in points (1" = 72 points)
  • Product height The height of the product in points (1" = 72 points)

Calculated values

Some of the data used for generating the laycrimp layout will be calculated based off the data collected.

  • Distance from side This is the number of points from the side of the sheet to edge of the first page encountered.
  • Distance from bottomThis is the number of points from the bottom edge of the sheet to the edge of the first page encountered
  • Product trim width The width of the final product after trim occurs
  • Product trim height The height of the final product after trim occurs

Determining the distance from the side

This value can be found by taking the width of the sheet and subtracting the total width of all products contained in a row, plus the sum of all horizontal gutters, multiplied by the total number of columns minus one, and divide this result by two. Written as an expression, it would look like the this. (This configuration assumes that autocenterx="true" and autocentery="true" have been defined in the imposition.)

((sheetWidth - (productWidth * numCols) + (hGutter * (numCols -1)) / 2

Determining the distance from the bottom

This value can be found by taking the height of the sheet and substracting the total height of all products contained in a column, plus the sum of all vertical gutters, multiplied by the total number of rows minus one, and divide this result by two. Written as an expression, it would look like this.

((sheetHeight - (productHeight * numRows) + (vGutter * (numRows - 1)) / 2

Determining the product trim width

The final product trim width will be determined by taking the product width and subtracting the amounts specified for trim left and trim right. Written as an expression, it would look like this.

productWidth - (trimLeft + trimRight)

Determining the product trim height

The final product trim height will be determined by taking the product height and subtracting the amounts specified for the trim top and trim bottom. Written as an expression, it would look like this.

productHeight - (trimTop + trimBottom)
