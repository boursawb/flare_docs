Updating the imposition

There are a few requirements and recommendations that should be followed when building out the batch imposition for a Smart Stacker integration.

Setting imposition attributes

  1. The imposition attribute reverseorder="true/false" must be set to true
  2. The attribute autocenterx and autocentery should both be set to true
  3. The attribute spreadwidth= should be set to the width of the sheet used for this product
  4. The attribute spreadheight= should be set to the height of the sheet used for this product
  5. Ensure that type has been set to stepandrepeat
  6. Save changes to the batch imposition

It is critical that the paper size does not change from what is originally defined when implementing a finishing plan with Horizon SmartStacker. This will result in unexpected and undesirable behavior during the finishing process. Changing the paper size will require that the dimensions defined in the imposition, device settings in the batch
