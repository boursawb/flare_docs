Designer mode

provides a way to convert an existing order in the system to designer mode for the purposes of viewing complex order structure and signature placement. This feature allows you to easily see which signatures belong together when viewing your output. Designer mode converts the order content to light-weight PDFs with each order having its own color for all pages within the order. Once the product design process is complete, the order can be converted back to its original form, ready for press submission.

Change an order to Designer Mode
  1. Right-click on the order to be changed
  2. Select Designer from the menu
  3. Select Mark Order For Designer Mode from the sub-menu
 Change an order back to Normal Mode
  1. Right-click on the order to be changed
  2. Select Designer from the menu
  3. Select Un Mark Order For Designer Mode from the sub-menu
