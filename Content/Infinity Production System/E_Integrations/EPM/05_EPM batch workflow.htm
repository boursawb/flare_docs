EPM batch workflow

Provided that EPM eStop is disabled, sort groups will automatically split by the EPM_Eval order attribute. For each batch created, EPM_Eval will roll up as a batch level attribute. By enabling this attribute on the batches tab, you can see which batches are enabled for EPM and which are not. From the batches tab, there are three menu options that can be used for testing EPM on press.

  • Mark For EPM on Reprint
  • Mark For EPM BW on Reprint
  • Remove EPM on Reprint

Testing EPM on press

To toggle EPM on/off for the selected batch, follow the steps shown below.

  1. Right-click on the batch
  2. Navigate to the EPM menu
  3. Select the desired option (Mark for EPM, Mark for EPM BW, or Remove EPM on reprint)
