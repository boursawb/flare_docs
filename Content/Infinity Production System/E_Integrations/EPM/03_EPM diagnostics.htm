EPM diagnostics

order history includes detailed information regarding EPM evaluations. This information can be used to understand and explain the results for each component evaluation.

To view EPM diagnostics information

  1. Identify an order on the Products tab to check EPM results
  2. Right-click on the order
  3. Navigate to View and select View Order History
  4. Filter the results by typing EPM in the filter criteria window

EPM events and event properties

Once EPM evaluation completes, detailed results of the evaluation can be found in order and EPM history.

  • Order EPM Queued occurs when the order is queued for the EPM evaluation process
  • Order EPM Start indicates when the EPM evaluation begins
  • Order Level EPM Evaluation Result displays detailed information regarding the results of the evaluation
  • Order EPM Task Completed shows when the EPM evaluation process completes
  • Order EPM Task Deleted shows when the EPM evaluation task is deleted

Understanding Order Level EPM Evaluation Result

The order level evaluation result include 3 event messages that provide the details of the evaluation process.

  • epmEvalResult displays when the component is deemed EPM compatible and will be either EPM or EPM_BW
  • epmEvalResultReason briefly explains why the order is not EPM compatible
  • epmEvalResultReasonDetails provides a detailed explanation as to why the order is not EPM compatible

The event messages displayed within the Order Level EPM Evaluation Result event will vary based on the evaluation results. When an order is found to be EPM compatible, only the epmEventResult will display. If an order is not EPM compatible, only the two result reason messages will be displayed.

Viewing order history EPM results

To view EPM results for a specific order, follow the steps shown below.

  1. Right-click on the order and select View Order History
  2. Filter the results by epm to show only relevant events

Viewing batch history EPM results

To view EPM results for a specific batch, follow the steps shown below.

  1. Right-click on the batch and select View Batch History
  2. Filter the results by epm to show only relevant events

Viewing EPM history

To review the results for all EPM evaluations, follow the steps shown below.

  1. .Select Tools in the menu at the top of the UI
  2. Navigate to Enhanced Productivity Mode
  3. Select View EPM History
  4. Filter for and view the items of interest
