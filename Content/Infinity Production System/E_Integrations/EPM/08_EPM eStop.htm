EPM emergency stop (eStop)

EPM eStop provides a way to turn off EPM in one or more steps of the workflow. There are three groups of work that can be addressed when determining which eStop options to use.

  • Order awaiting EPM evaluation
  • Work queued to Enfocus
  • Orders tagged EPM compatible, ready to be released / batched

The three options available via EPM eStop applicable to these groups of work are shown below.

  • Disable Order EPM Evaluation: When checked, no orders are submitted for EPM evaluation and any existing work queued in EPM evaluation is canceled.
  • Disable Sort Group EPM Processing: When checked, sort groups are no longer split by EPM evaluation status and all work is released as non EPM. A reprocess at sort must be performed on the affected sort groups following any changes to the sort group workflow of the ESTOP configuration.
  • Disable Batch EPM Processing: When checked, EPM attributes of batches are ignored and the batch submitted to the DFE as a normal / non-EPM job.

When EPM eStop is invoked, attributes applied to orders identifying them as EPM compatible will not be removed. If / when EPM is turned back on, orders already evaluated will be available for EPM printing with no user action required.
