System configuration

To enable the SDK event workflow, the system configuration must be updated. The process will involve updating the workflow event and job status plugin configurations.

Update workflow event plugin

  1. Browse to the sharedconfig location
  2. Open the worflowevents.plugins.xml file located in the root of sharedconfig
  3. Add the following entry to the pluginClassNames section

<!-- SDK -->
<string>com.infdig.idf.sdk.SDKWorkflowEventsBridgePlugin</string>

Update the job status plugin

  1. Browse to the sharedconfig location
  2. Open the dfejobstatus.plugins.xml file located in the root of sharedconfig
  3. Add the following entry to the pluginClassNames section

<string>com.infdig.idf.sdk.SDKWorkflowEventsBridgePlugin</string>

Restart services

Once the new plugin updates have been saved, all distributed core services (DCore) will need to be restarted.
